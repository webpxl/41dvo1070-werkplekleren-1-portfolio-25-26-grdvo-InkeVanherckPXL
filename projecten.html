Schoolprojecten

Tijdens mijn opleiding Digitale Vormgeving aan PXL heb ik gewerkt aan diverse opdrachten waarbij creativiteit en techniek samenkomen.

UI Design

Voor dit project heb ik een nieuwe app moeten ontwerpen. Ik heb een app gemaakt 'Snacktrack' waarbij je wandelingen moet voiltooien om kortingen te verzamelen voor restaurants. Ik heb hiervoor sketches, sitemap, taskflows, wireframe, UI-kit, visuals design en prototype moeten maken. Hiervoor moesten ook een website designen voor zowel desktop als mobile. Dit project hield mij creatief bezig.

Tools: Figma, Adobe Illustrator

Dashboard cv

Bij het vak werkplekleren heb ik een dashboard cv moeten ontwikkelen op figma. Dit werd ontwikkeld stapgewijs in vervulling gebracht. Eerst bepalen wat ik in mijn dashboard cv wil en insperatie opdoen bij andere cv's. Daarna maakte ik schetsen en bepaalde ik welke componenten ik statisch of dynamisch wou en maakte ik hierna wireframes van. Hierna zocht maakte ik een moodboard voor het design van mijn dashboard cv; Uiteindelijk na veel opties proberen is mijn dashboard cv tot een eindstand gekomen. Bij deze processen kreeg ik ook bij elke stap feedback van klasgenoten over wat er beter kon of wat al goed was.

Tools: Figma

Scripting

Tijdens het vak Scripting heb ik een take-away website ontwikkeld met de naam Delizinke. Deze website heb ik gemaakt met behulp van HTML, CSS en JavaScript. Binnen dit project heb ik gewerkt aan zowel de vormgeving als de functionaliteit van de website, waardoor gebruikers eenvoudig bestellingen kunnen bekijken en plaatsen.

Tools: HTML, CSS, Scripting (webstorm)
