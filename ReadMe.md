Ik heb een porfolio website gemaakt met als beginpagina een overzicht van mijn profiel, mijn studiekeuze, mijn diploma's, werkervaring,
skills, taalvaardigheden en nevenactiviteiten.

Bovenaan in de navigatiebalk kan je navigeren naar mijn projecten en naar weetjes van mezelf.

overal is een footer waarbij je mijn privacyverklaring kan openen.

Als bronnen heb ik gebruik gemaakt van mijn eigen figmafiles, unsplash voor de afbeeldingen, de cursus van web en google.gemini en chatgpt 
voor extra hulp bij het coderen.
